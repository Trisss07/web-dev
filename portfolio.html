My Projects

My Study Planner

Overview:

The Study Planner App is a simple, beginner‑friendly web tool designed to help students organise their workload more effectively. It allows users to list tasks, set deadlines, and keep track of upcoming assignments in a clean, minimal interface.

What it does:

  • Lets students add, edit, and remove tasks
  • Displays deadlines in an easy‑to‑read list
  • Helps with time management and planning

Why I built it:

As a first‑year CS student, I wanted a tool that could help me stay organised while learning new concepts. This project helped me practise structuring a webpage, working with forms, and designing a user‑friendly layout.

Grid Game

Overview:

Grid Adventure Game is a small Java project where the player moves around a grid, collects points, and avoids traps. It’s a simple logic‑based game that demonstrates movement, scoring, and basic game mechanics.

What it does:

  • Allows the player to move up, down, left, and right
  • Generates traps and point tiles on the grid
  • Ends the game when a trap is hit

Why I built it:

This project was part of my coursework and helped me understand Java fundamentals like loops, arrays, user input, and game logic. It also taught me how to break a problem into smaller parts and build it step by step.

Winter Scene Animation

Overview:

The Winter Scene Animation is a creative web project built using HTML and CSS animations. It features a cosy winter landscape with falling snow, moving elements, and smooth transitions.

What it does:

  • Uses CSS keyframes to animate snowflakes
  • Includes layered scenery for depth
  • Creates a calm, festive atmosphere

Why I built it:

I enjoy making visually appealing web pages, and this project let me experiment with animation timing, positioning, and layering. It helped me understand how small visual details can make a webpage feel alive.
